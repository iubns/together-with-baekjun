## 문제

> 준하가 가입하려고 하는 사이트에 이미 존재하는 아이디면 뒤에 `??!`를 붙여서 새 아이디를 만든다.  
입력으로 준하가 가입하려고 하는 아이디가 주어졌을 때, 새로운 아이디를 출력하는 프로그램을 작성하시오.

## 입력

> 첫째 줄에 준하가 가입하려고 하는 아이디가 주어진다.  
아이디는 알파벳 소문자로만 이루어져 있고, 길이는 50자를 넘지 않는다.

## 출력

> 첫째 줄에 준하의 새 아이디를 출력한다.

## 예제 입력

> joonas

## 예제 출력

> joonas??!
