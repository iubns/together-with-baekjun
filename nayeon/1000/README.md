## 문제

> 두 정수 A와 B를 입력받은 다음, A+B를 출력하는 프로그램을 작성하시오.

## 입력

> 첫째 줄에 A와 B가 주어진다. (0 < A, B < 10)

## 출력

> 첫째 줄에 A+B를 출력한다.

## 예제 입력

> 1 2

## 예제 출력

> 3

## 💭 풀이 과정 (ETC)

- `Scanner`로 입력을 받고, `System.out.println()`으로 출력 함


## 🔥 풀이 핵심:  
- `Scanner sc = new Scanner(System.in);` 입력 
- `int A = sc.nextInt();`  
- `System.out.println(A + B);` 출력
