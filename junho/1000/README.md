## 문제 (Problem)

> 두 정수 A와 B를 입력받은 다음, A+B를 출력하는 프로그램을 작성하시오.

## 입력 (Input)

> 첫째 줄에 A와 B가 주어진다. (0 < A, B < 10)

## 출력 (Output)

> 첫째 줄에 A+B를 출력한다.

## 예제 입력 (Exercise)

> 1 2

## 예제 출력 (Output)

> 3
